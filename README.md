NPang
=====
더치페이 계산기(엔빵) 


Features
========
- 과거 비용지출 이력 관린
- 참석자별 입금 여부 관리
- 입금 여부에 따른 자동 알림 설정
- 계좌번호 자동 저장
- 카톡, 라인, 마플 등 영수증 발송 지원 


Dependencies
============
* AndroidAnnotations
* ActionbarSherlock
* EventBus
* OrmLite
* NineOldAndroids
* UrlImageViewHelper
* AndroidBootstrap
* AndroidHorizontalListView
